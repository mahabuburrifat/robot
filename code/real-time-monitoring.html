REAL TIME ENVIRONMENT MONITOR
AND ALERM SYSTEM
(INTERNET OF THINGS)

THIS PROJECTS CAN SENSE TEMPARATURE, HUMIDITY, AIR QUALITY AS WELL AS DETECT FLAME. WHILE ANY ABNORMAL CONDITION IN ENVIRONMENT, THIS SYSTEM CAN SEND NOTIFICAITON VIA E-MAIL, PHONE CALL AS WELL AS SMS.

This environment monitoring and alarm system is built using ESP-NOW protocol and microcontrollers like ESP32 and ESP8266. The system continuously monitors the temperature, humidity, and smoke levels in your environment, and sends real-time data to the cloud using wireless communication protocols through ESP-NOW as well as send the whole information through MQTT.
